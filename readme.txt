#NSQL Ver 1.0
#使い方説明書

##ライセンス
NSQLはMIT ライセンスです。

##ファイル組成
*data/
データベースファイルを保存するディレクトリです。
*data/.htaccses
データベースファイルが一般公開されるのを防ぎます。このファイルを消すとデータが一般からアクセスされ、データが危険にさらされる可能性があります。
*nsql.php
データベース書き込み/読み込みを担当するファイルです。
他はなくても最低限動きますが、これがないと絶対に動きません。

##仕様詳細
データはjson形式で保存されます。

##サンプルコード
<?php
$db = new nsql('/data/test.json');

$text['朝のデータ'] = 'おはよう。';
if($db->edit($text)){
 $text = $db->view();
 echo '反映されました。';
}else{
 echo 'エラー:データベースファイルに書き込むことができませんでした。';
}

$text = $db->view();
echo $text['朝のデータ'];

$text['朝のデータ'] = 'いただきます。';
if($db->edit($text)){
 $age = $db->view();
 echo '反映されました。もう朝ごはんの時間です！';
}else{
 echo 'エラー:データベースファイルにアクセスできませんでした。まだ寝てていいよ。';
}
?>

一度このファイルを動作させて、動作確認してから使ってください。
動作しない場合は最初に、[データベース名].json ファイルの中に {} というコードをUTF-8/Domなしで追加してください。
